Crafteo, ¿Qué es?

El crafteo es la elaboración de objetos nuevos apartir de diversos materiales. Todo esto se realiza mediante una mesa de crafteo o mesa de trabajo.

Básicos
[Indice]


Madera refinada: Usado para construcciones de todo tipo, puede usarse para elaborar multitud de cosas.


Mesa de crafting: Permite acceder al menu de elaboración de 3x3.


Palos: Usado para elaborar antorchas, flechas, carteles, escaleras, vallas y mangos de herramientas.


Horno: Permite al jugador Fundir o cocinar cosas.


Cama: Sirve para dormir, al dormir transcurre la noche rápidamente
Antorchas [Indice]


Jack-O-Lantern: se utiliza como elemento de decoración. Posee la misma iluminación que una antorcha. Si se unde en el agua, esta sigue brillando.


Antorchas: Usado para crear luz. Las antorchas también funden la nieve y el hielo.


Antorcha de RedStone: Envía cargas electricas. Tambien puede ser usado para una antorcha de baja iluminación.
Cofres [Indice]


Cofre trampa:. Sirve para elaborar trampas.


Cofre Enderman: permite que lo que este guardado en uno pueda ser obtenido de otro Cofre de Ender sin importar la distancia o dimension en que se encuentre.


Cofre: Permite guardar bloques y objetos dentro.
Escaleras [Indice]


Escaleras de pared: sirven para subir por un plano inclinado.


Escalones: Usado para hacer escaleras completas. La utilización permite al jugador subir sin saltar. Pueden ser de piedra, ladrillo de piedra, ladrillo, arenisca, madera de roble, abedul, de jungla, de taiga, de roble oscuro, ladrillos abisales, cuarzo.
Losas [Indice]


Losas: Se usa para hacer escaleras largas. Dos escalones colocados, uno sobre, otro produce un bloque completo. Pueden ser de piedra, ladrillo de piedra, madera de roble, abedul, de jungla, roble oscuro, taiga, arenisca, ladrillos, ladrillos abisales, cuarzo.
Tintes [Indice]


Tintes: Utilizado para colorear lanas,cristales o bloques de cera endurecidos. Hay de distintos colores. Se crea utilizando flores, plantas y otros items.
- Crafteos con tintes [Indice]
Tinte de lanas [Indice]


Lana tintada: Todas las lanas, siguen el procedimiento de coloreado que se muestra


Alfombra tintada: Todas las alfombras, siguen el procedimiento de coloreado que se muestra


Cristal tintado: Todos los cristales, siguen el procedimiento de coloreado que se muestra
Redstone [Indice]
- Placas [Indice]


Placas de piedra: permite accionar circuitos poniéndonos encima de ella.


Placas de madera: permite accionar circuitos poniéndonos encima de ella.


Placas de hierro: permite accionar circuitos poniéndonos encima de ella o bién mediante el peso de varios objetos encima de ella.


Placas de hierro: permite accionar circuitos poniéndonos encima de ella o bién mediante el peso de varios objetos encima de ella.
- Botones/Palancas [Indice]


Palanca: Envía cargas eléctricas, se turna en Encendido o Apagado.


Botón de piedra: se utiliza para activar circuitos con RedStone, o objetos cercanos, como por ejemplo una puerta.


Botón de madera: se utiliza para activar circuitos con RedStone, o objetos cercanos, como por ejemplo una puerta. A fiderencia de el de piedra, éste se puede activar mediante el contacto con una flecha.
- Puertas [Indice]


Puerta de madera: Las puertas de madera pueden ser abiertas mediante un Click. Esta puertas pueden ser destruidas por zombies, mediante un continuo ataque, no son muy resistentes en comparación a las de hierro.


Puerta de acero: Las puertas de Acero pueden ser abiertas por un interruptor y/o Carga electrónica, pero nunca como las puertas de madera, mediante un click. Son las mas resistentes y defensivas.
- Trampillas [Indice]


Trampilla de madera: Sirve para permitir o denegar el paso, puede accionarse mediante palancas, botones e incluso click derecho, sin ninguna de las anteriores.


Trampilla de hierro: accionable solo mediante el uso de palancas o botones.
- Circuitos [Indice]


Repetidor de RedStone: Funciona como renovador de la corriente eléctrica.


Comparador de RedStone: Divide las cargas elécticas, según queramos.
- Vías [Indice]


Raíles: Permiten que pueda circular los minecarts.


Raíl con placa: Se utiliza para hacer reaccionar circuitos con RedStone.


Raíl impulsador: se utilizan para impulsar los minecarts.


Raíl: Se utiliza para impulsar minecarts, mediante el uso de botones o palancas, como accionadores.


Minecart: Sirve para transportar jugadores.


Minecart con horno: utilizado como combustible para impulsar a los minecarts por railes sin redstone.


Minecart con confre: Utilizado para transportar objetos.


Minecart con TNT: Utilizado para ser detonado, puede activarse mediante un raíl detector, o un simple encendedor.


Minecart con Embudo: Utilizado para recojer materiales lanzados por un soltador.
- Otros [Indice]


Reloj: Usado para ver cuanto tiempo transcurre entre el día y la noche.


Bloque de notas: Toca notas musicales. Haz click izquiero para tocar una misma nota. Click derecho para cambiar el tono. Nota: El sonido de el bloque de notas puede variar sobre que superficie este colocado.


Pistón: Empuja un bloque. Funciona mediante un interruptor y/o una carga eléctrica.


Pistón pegajoso: Funciona de misma manera que un pistón regular, pero el bloque queda adherido al pistón.


Lámpara de RedStone: Funciona como una lámpara.


Dispensador: El dispensador mantiene objetos en casillas de 3 x 3. Cuando es utilizado con Polvo de Piedra Roja, éste lanza objetos como huevos, bolas de nieve, e incluso flechas como si fuese un arco.


Soltador: Utilizado para lanzar materiales depositados dentro de él, mediante palancas o otros circuitos.
Bloques [Indice]
- Básicos [Indice]


Bloque de GlowStone: Tiene las mismas propiedades de luz que los bloques naturales. Romperla devuelve una sola unidad de energía. Nota: Este bloque actúa como una antorcha, y fundirá la nieve y el hielo.


Lana: Usado como material de construcción. La tela puede ser obtenida también de las ovejas.


Bloque de nieve: Se usa para almacenar bolas de nieve o como material de construcción, también es usado en la creación de golems de nieve.


Ladrillo de piedra: Es un nuevo material introducido en la Beta 1.8 que puede servir para construcción


Bloque de arcilla: Utilizado para almacenar arcilla, también puede ser usado como bloque de construcción.


Bloque de ladrillos: Utilizado como bloque decorativo.


Bloque de arenisca: Utilizado como bloque decorativo o para elaborar bloques de arenisca lisos.


Bloque de arenisca lisa: Utilizado como bloque decorativo.


Bloque de cuarzo: Utilizado como bloque decorativo.


Bloque de cuarzo cincelado: Utilizado como bloque decorativo.


Bloque de TNT: Se usa para provocar explosiones, solo 1 de cada 4 de los minerales destrozados con TNT, darán un bloque.


Bloque de carbón: Utilizado para almacenar carbón, puede ser usado como bloque decorativo o como fuente de energia en la cocción de materiales (horno).


Bloque de melón: Utilizado para almacenar melón, o puede ser usado como bloque decorativo.


Bloque de trigo: Utilizado para almacenar trigo , o puede ser usado como bloque decorativo.


Bloque de RedStone: Permite elaborar bloques a partir de lingotes o gemas de minerales, útiles para un almacenamiento más compacto.


Bloque de lapislazuli: Permite elaborar bloques a partir de lingotes o gemas de minerales, útiles para un almacenamiento más compacto.


Bloque de hierro: Permite elaborar bloques a partir de lingotes o gemas de minerales, útiles para un almacenamiento más compacto.


Bloque de oro: Permite elaborar bloques a partir de lingotes o gemas de minerales, útiles para un almacenamiento más compacto.


Bloque de diamante: Permite elaborar bloques a partir de lingotes o gemas de minerales, útiles para un almacenamiento más compacto.


Bloque de esmeralda: Permite elaborar bloques a partir de lingotes o gemas de minerales, útiles para un almacenamiento más compacto.


Minerales: una vez almacenados los minerales los podemos volver a cojer, poniendo el bloque en la mesa de crafteo.


Bloque de arenisca endurecido tintado: se usa como elemento decorativo.


Paneles de cristal y paneles de cristal tintados: pueden sustituir a los bloques de cristal en función de las ventanas, tienen una función decorativa.
